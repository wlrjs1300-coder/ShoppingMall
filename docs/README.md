# 문서 안내

따뜻한 떡집 프로젝트의 발표, 검토, QA용 문서를 모아둔 폴더입니다.

## 문서 목록

- `PROJECT_SUMMARY.md`: 프로젝트 개요, 구현 기능, ERP 4축 반영 내용, 남은 개선 후보
- `PRESENTATION_GUIDE.md`: 발표 순서, 설명 멘트, 예상 질문 답변
- `DEMO_SCENARIO.md`: 실제 화면을 클릭하며 보여줄 시연 순서
- `QA_CHECKLIST.md`: 발표 전 기능과 화면을 점검하는 체크리스트
- `QA_RESULT.md`: 정적 QA 점검 결과 기록
- `TECHNICAL_OVERVIEW.md`: 파일 구조, 데이터 저장 방식, 관리자 ERP 로직 설명

## 추천 사용 순서

1. `PROJECT_SUMMARY.md`로 전체 기능을 먼저 파악합니다.
2. `PRESENTATION_GUIDE.md`로 발표 흐름을 정합니다.
3. `DEMO_SCENARIO.md`를 보며 실제 클릭 순서를 연습합니다.
4. `QA_CHECKLIST.md`로 발표 전 최종 점검을 진행합니다.
5. `QA_RESULT.md`에서 마지막 정적 점검 결과를 확인합니다.
6. 기술 질문이 나오면 `TECHNICAL_OVERVIEW.md`를 참고합니다.

## 빠른 실행

루트 폴더의 `open-presentation-docs.cmd`를 실행하면 발표 관련 문서를 한 번에 열 수 있습니다.
