# 따뜻한 떡집 QA 체크리스트

## 기본 접속

- `index.html`이 정상적으로 열리는지 확인
- `menu.html`이 정상적으로 열리는지 확인
- `faq.html`이 정상적으로 열리는지 확인
- `admin.html` 접속 후 코드 `4525`로 관리자 화면이 열리는지 확인

## 고객 화면

- 상단 메뉴가 `홈 / 메뉴 / 자주 묻는 질문`으로만 보이는지 확인
- 로고 클릭 시 메인으로 이동하는지 확인
- 메인 화면의 `메뉴와 가격 보기` 버튼이 메뉴 페이지로 이동하는지 확인
- 메뉴 페이지에서 BEST 상품이 보이는지 확인
- 전체 메뉴 검색이 작동하는지 확인
- 카테고리 필터가 작동하는지 확인
- 메뉴 페이징이 작동하는지 확인
- 상품 클릭 시 상세 패널이 열리는지 확인
- 주문 상담 요청 패널이 열리는지 확인
- `배송 상담` 선택 시 배송 주소 입력이 강조되고 필수 입력되는지 확인

## 관리자 화면

- 관리자 확인 코드가 틀릴 때 안내 문구가 표시되는지 확인
- 관리자 운영 흐름 카드 클릭 시 해당 탭으로 이동하는지 확인
- 주문관리에서 주문 검색과 상태 필터가 작동하는지 확인
- 주문 직접 등록과 수정이 가능한지 확인
- 주문관리의 `운영 데이터 초기화`가 주문, 고객 메모, 직접 등록 고객을 함께 정리하는지 확인
- 고객관리에서 고객 등록, 수정, 삭제가 가능한지 확인
- 운영 알림 센터의 주문, 재고, 발주, 물류 대기 수가 실제 데이터와 일치하는지 확인
- 운영 알림을 클릭하면 해당 관리 탭으로 이동하는지 확인
- 주문, 생산, 재고, 발주 변경 후 최근 활동 로그가 생성되는지 확인
- 최근 활동을 클릭하면 관련 관리 탭으로 이동하고 기록 비우기가 작동하는지 확인
- 생산관리에서 날짜별 생산 수량이 표시되는지 확인
- 생산 항목을 `준비완료`로 일괄 처리할 수 있는지 확인
- 부족·주의 재고가 발주 후보에 표시되는지 확인
- 권장 발주량으로 발주 요청이 등록되는지 확인
- 발주 상태를 `발주중`, `입고완료`로 변경할 수 있는지 확인
- 공급처와 단가 입력 후 품목별 발주금액과 진행 발주액이 계산되는지 확인
- 공급처 등록, 수정, 삭제와 발주 입력 자동완성이 작동하는지 확인
- `입고완료` 처리 후 현재 재고와 상태가 갱신되는지 확인
- 픽업/배송 관리에서 물류 상태를 변경할 수 있는지 확인
- 물류 상태를 `완료`로 바꾸면 주문 상태도 `픽업완료`로 바뀌는지 확인
- 매출관리에서 매출, 원가, 이윤, 이윤율이 계산되는지 확인
- 매출관리에서 입고완료 발주비와 공급처별 비용이 집계되는지 확인
- 총매출에서 입고 발주비를 차감한 순현금흐름이 표시되는지 확인
- 월별 매출, 예상이윤, 발주비, 순현금흐름 추이가 표시되는지 확인
- `회계 CSV` 저장 파일에 월, 매출, 원가, 이윤, 발주비, 순현금흐름이 포함되는지 확인

## 데이터

- 메뉴 페이지에서 남긴 상담 요청이 관리자 주문관리로 들어오는지 확인
- 고객 정보가 고객관리 탭에 자동 집계되는지 확인
- 배송 상담 주문이 픽업/배송 탭에서 배송 상담으로 표시되는지 확인
- CSV 저장이 작동하는지 확인
- JSON 백업 저장이 작동하는지 확인
- JSON 백업 불러오기가 작동하는지 확인

- `docs/PRESENTATION_GUIDE.md`에서 발표 순서와 질문 대응을 확인
- 화면 확대/축소 100% 기준으로 주요 UI가 겹치지 않는지 확인
