# 따뜻한 떡집 프로젝트 정리

## 프로젝트 개요

`따뜻한 떡집`은 떡 판매용 홈페이지와 간단한 관리자 ERP 화면을 함께 구성한 초기 쇼핑몰 프로젝트입니다. 고객은 메뉴를 확인하고 주문 문의를 남길 수 있으며, 관리자는 주문, 고객, 생산, 픽업/배송, 매출 정보를 한 화면에서 관리할 수 있습니다.

## 고객용 페이지

- `index.html`: 브랜드 첫 화면, 대표 문구, 주요 이동 카드
- `menu.html`: BEST 상품, 전체 메뉴, 검색/필터, 페이징, 상품 상세, 주문 문의
- `faq.html`: 주문, 예약, 단체주문, 픽업/배송 관련 자주 묻는 질문

## 관리자 페이지

- 접속 파일: `admin.html`
- 임시 확인 코드: `4525`
- 운영 흐름 카드: 주문 접수 -> 생산 준비 -> 픽업/배송 -> 매출 확인
- ERP 구성 카드: 회원, 생산, 물류, 회계 영역을 첫 화면에서 바로 확인하고 각 관리 탭으로 이동
- 운영 알림 센터: 접수대기 주문, 부족 재고, 진행 발주, 픽업·배송 대기를 실시간 집계하고 관련 탭으로 이동
- 최근 활동 로그: 주문, 생산, 재고, 발주·입고 주요 변경을 최대 100건 저장하고 관련 탭으로 이동
- 테스트 데이터: `admin.html?dev=1` 접속 시 데모 생성 버튼 표시

## ERP 구성 반영

관리자 첫 화면에서 회원, 생산, 물류, 회계 카드를 제공해 쇼핑몰 운영에 필요한 4가지 관리 영역을 한눈에 확인할 수 있습니다. 각 카드는 해당 관리 탭으로 연결되어 시연할 때 구현 범위를 빠르게 보여줄 수 있습니다.

### 인사/고객 관리

- 주문 고객 자동 집계
- 고객 직접 등록, 수정, 삭제
- 고객 연락처, 구분, 메모 관리
- 주문 이력 기반 고객별 주문 횟수, 수량, 매출 확인

### 회계 관리

- 주문별 판매가, 원가, 이윤 입력 및 계산
- 상품별 판매수량, 주문건수, 매출, 원가, 이윤, 이윤율 집계
- 일자별 매출 요약
- 매출 1위 상품, 이윤 1위 상품, 원가 미입력 건수 확인
- 입고완료일 기준 발주비 집계
- 공급처별 입고 발주비와 순현금흐름 확인
- 최근 12개월 매출, 예상이윤, 발주비, 순현금흐름 추이
- 현재 회계 기간의 월별 보고서 CSV 저장
- CSV 백업 및 JSON 백업/복원

### 물류 관리

- 고객 주문 문의에서 매장 픽업/배송 상담 선택
- 배송 상담 선택 시 배송 주소 입력 요구
- 관리자에서 픽업/배송 상태 관리
- 물류 상태 완료 처리 시 주문 상태도 픽업완료로 연동

### 생산 관리

- 픽업일 기준 생산 목록 집계
- 상품별 준비 수량, 주문건수, 픽업시간 확인
- 오늘 생산 수량 표시
- 상품/날짜 묶음 단위로 준비완료 일괄 처리
- 준비완료 처리 시 대표 원재료 재고 자동 차감

### 재고 관리

- 원재료 현재 재고, 단위, 안전 재고 등록
- 부족/주의/정상 상태 자동 분류
- 생산관리 예상 원재료와 연동
- 상품별 원재료 기준표 표시, 추가, 수정, 삭제
- 기본 재고 샘플 생성
- 생산 완료 시 원재료 사용 이력 자동 기록
- 부족·주의 품목의 권장 발주량 자동 계산
- 발주 후보에서 권장 수량으로 발주 요청 생성
- 발주요청, 발주중, 입고완료 상태 관리 및 완료 시 재고 반영
- 공급처, 단가, 품목별 발주금액과 진행 발주액 관리
- 공급처 등록, 수정, 삭제와 연락처·취급 품목·거래 메모 관리
- 발주 현황 공급처 자동완성 연동
- 재고 데이터 백업/복원 포함

## 시연 순서 추천

1. `index.html`에서 브랜드 첫 화면 확인
2. `menu.html`에서 메뉴 검색, 필터, 페이징 확인
3. 메뉴 상품을 눌러 주문 문의 작성
4. `admin.html` 접속 후 코드 `4525` 입력
5. 주문관리에서 방금 등록한 주문 확인
6. 생산관리에서 준비 수량 확인 후 준비완료 처리
7. 픽업/배송 관리에서 물류 상태 완료 처리
8. 매출관리에서 판매가, 원가, 이윤 흐름 확인
9. 고객관리에서 고객 정보와 메모 확인

## 현재 한계

- 정적 HTML 기반이라 실제 서버 로그인 보안은 아직 없습니다.
- 주문 데이터는 브라우저 `localStorage`에 저장됩니다.
- 네이버 스마트스토어 실제 연동은 아직 연결 전입니다.
- 실제 결제, 배송 송장, 재고 차감, 권한 관리는 추후 서버와 DB가 필요합니다.
