# 메뉴 이미지 파일명 매핑

아래 파일을 `assets/products` 폴더에 같은 이름으로 넣으면 `menu.html`의 각 메뉴 카드에 바로 반영됩니다.

## 1순위 적용 완료

- 송편 예약, 깨송편: `menu-songpyeon.png`
- 백일떡: `menu-baekil.png`
- 수수팥떡: `menu-susupat.png`
- 답례떡: `menu-gift-box.png`
- 단체주문: `menu-bulk-order.png`
